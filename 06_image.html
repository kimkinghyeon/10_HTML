이미지 관련 태그


src 속성에 경로를 설정하여 이미지 파일을 부러온다.

자주사용하는 속성들

alt 속성

사진의 경로가 잘못되거나, 이미질를 제댈로 표현 할 수 없을경우 대체 텍스트 용도로 사용된다.


width , hight 속성

    사진의 높이와 너비를 지정 할 수 있다.
    고정된 단위의 크기로도 지정 가능하며
    가변크기 단위의 크기로도 지정이 가능하다.

고정 크기 단위 : 화면 사이즈가 줄어도 크기가 변하지 않는다.

가변 크기 단위 : 화면 사이즈가 줄어도 크기가 변하지 않는다.
